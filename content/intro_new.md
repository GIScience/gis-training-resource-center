# IFRC GIS Training Platform

:::{card}
## GIS training for humanitarian and anticipatory action
Welcome to the **IFRC Network GIS Training Platform**.

This platform provides **training materials, tools, and applied examples** to support the use of **QGIS in the Red Cross Red Crescent Network**, with a strong focus on **Anticipatory Action** and humanitarian decision-making.

The content is designed primarily for **GIS trainers**, but also supports practitioners and technical staff who work with geospatial data in operational contexts.
:::

---

## How to Use This Platform

::::{grid} 2

:::{card} Start teaching
:link: https://giscience.github.io/gis-training-resource-center/content/Trainers_corner/en_TOT_intro.html
This platform is primarily designed to support **GIS trainers** in the IFRC Network.

You can:
- Use **training plans** to structure complete courses
- Combine **modules, exercises, tools, and case studies**
- Adapt materials to different operational and country contexts

All trainer-specific guidance is collected in the **Trainer’s Corner**, which is part of the *Learn* section.
:::

:::{card} Start learning
:link: https://giscience.github.io/gis-training-resource-center/content/Module_1/en_module_1_overview.html

This is **not a self-learning platform**.

Learning is most effective when guided by an experienced trainer.  
Modules and exercises are designed to support **instructor-led trainings**, but can also be revisited to refresh and consolidate skills.

Your trainer will guide you through the appropriate learning path.
:::

## What You’ll Find on the Platform

The platform is organised around **three main pillars**, reflecting how the content is used in practice.

::::{grid} 3

:::{card}
### 🎓 Learn
**Build GIS knowledge and teaching capacity**

- Learning modules (basic to advanced)
- Hands-on exercises
- GIS Wiki and glossary
- Guidance for trainers and training plans

This pillar contains everything needed to **teach and learn GIS with QGIS**.
:::

:::{card}
### 🌍 Apply
**Use GIS in real humanitarian contexts**

- GIS in Anticipatory Action
- Country- and hazard-specific case studies
- Applied analyses and workflows
- Exercises focused on operational use cases

This pillar shows **how GIS supports real-world decision-making**.
:::

:::{card}
### 🛠️ Tools & Methods
**Reusable tools and workflows**

- QGIS plugins and tools
- Method-specific documentation
- Tool-focused tutorials and guidance

These tools are often used in applied case studies, but are documented separately to support **reuse across contexts**.
:::

## Learning Pathways (Modules Overview)

The learning content is organised into **three progressive skill levels**, each consisting of three modules.

:::{dropdown} 🟢 Basic Skills
| Module 1 – Introduction to GIS | Module 2 – Working with Geodata | Module 3 – Basic GIS Operations |
|---|---|---|
| GIS fundamentals | Vector & raster basics | Editing & selection |
| QGIS interface | Data sources | Attribute queries |
| Maps & cartography | Projections | Spatial queries |
| WebGIS & mobile apps | Importing data | Classification |
:::

:::{dropdown} 🟡 Intermediate Skills
| Module 4 – Representation | Module 5 – GIS Operations | Module 6 – Data Analysis |
|---|---|---|

| Atlases | Attribute operations | |
:::

:::{dropdown} 🔴 Advanced Skills
| Module 7 – Automation | Module 8 – Remote Sensing | Module 9 – Accessibility |
|---|---|---|
| Model builder | Raster analysis | Accessibility analysis |
| Automation concepts | Zonal statistics | Spatial analysis |

## GIS in Anticipatory Action

:::{card}
### Applying GIS to anticipatory decision-making
The **GIS in Anticipatory Action** section demonstrates how GIS is used to support early action before disasters occur.

It includes:
- Country- and hazard-specific case studies (e.g. cyclones, droughts, floods)
- Trigger and impact analyses
- Applied workflows using QGIS and specialised tools

This section connects **learning content** with **operational practice**.
:::
