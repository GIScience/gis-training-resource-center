# Procesamiento de datos

QGIS le permite procesar y modificar los conjuntos de datos cargados en su QGIS de diversas maneras. En general, existen dos tipos de procesamiento: espacial y no espacial. El procesamiento espacial modifica los datos en función de su relación espacial con las entidades de otras capas. El procesamiento no espacial modifica los datos en función de los valores de su tabla de atributos.

:::{Admonition} Tenga cuidado al modificar los conjuntos de datos originales.
:class: Tip

Si modifica la tabla de atributos o las entidades de un conjunto de datos que importó a su proyecto QGIS, el conjunto de datos original se modificará. Para asegurarse de mantener el conjunto de datos original intacto, asegúrese de crear una copia antes de realizar cualquier cambio.

:::
