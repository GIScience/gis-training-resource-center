# Opérations SIG de base <a id="basic-gis-operations"></a>

Le troisième module traite des opérations et des manipulations de données que les praticien·ne·s SIG réalisent fréquemment. Le premier chapitre présente le processus de numérisation, c’est-à-dire la création de données géographiques à partir d’images ou de cartes existantes, ou encore à partir de vos propres connaissances. Après la collecte de données, la numérisation permet de préparer les données collectées pour une analyse spatiale dans QGIS. Elle est également utile pour communiquer la répartition spatiale de phénomènes ou simplement pour afficher les données collectées sur une carte. Le deuxième chapitre se concentre ensuite sur la classification des données géographiques. Les données spatiales peuvent contenir une grande quantité d’informations dans leur table attributaire, et la classification permet d’organiser ces informations en classes distinctes. Grâce à cette classification, nous pouvons obtenir de nouvelles informations et visualiser les données géographiques sur des cartes. Le troisième chapitre porte sur la sélection et les requêtes appliquées aux données géographiques. Par exemple, vous pouvez sélectionner toutes les entités situées à moins d’une certaine distance d’un point, ou encore sélectionner des données en fonction de valeurs spécifiques et en observer la répartition spatiale.
