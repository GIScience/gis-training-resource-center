# Exercices du module 2 <a id="exercises-for-module-2"></a>

Ces exercices sont conçus pour accompagner une formation QGIS et peuvent être réalisés avec les participant·e·s en mode pas-à-pas ou comme travail à faire avant la séance suivante.  
- Les exercices 1 et 2 couvrent les bases afin de familiariser les débutant·e·s avec les données géographiques ainsi qu’avec l’import, l’exploration et l’enregistrement de données vectorielles dans QGIS. Ces exercices se prêtent bien à des séances en mode pas-à-pas.  
- Les exercices 2 et 3 prolongent les deux premiers exercices et visent à apprendre aux débutant·e·s en SIG à rechercher, télécharger et importer des données depuis le web.  
- L’exercice 5 récapitule le module 2 et peut être utilisé par les participant·e·s pour rafraîchir __de manière autonome__ leurs compétences sur QGIS avant la séance suivante.

| Exercise| Description |Focus Group|Estimated time| 
| :-------------------- | :----------------- |:----------------- |:----------------- |
|__[Exercice 1 : Comprendre les données géographiques](/content/fr/Module_2/fr_qgis_geodata_concept_ex1.md)__ | L’objectif de cet exercice est de faire vos premiers pas dans QGIS. Comprendre l’interface utilisateur et se familiariser avec le concept de couche. Afficher des données vectorielles dans QGIS et consulter leurs attributs. Reprojeter les données vectorielles (c’est-à-dire modifier leur projection). | Fondamentaux QGIS | 2 heures | 
|  __[Exercice 2 : Traitement de base des données géographiques](/content/fr/Module_2/fr_qgis_basic_data_processing_ex2.md)__ | L’objectif de cet exercice est de se familiariser avec les données géographiques et de commencer à les manipuler. Comprendre la table attributaire, la trier, effectuer des sélections manuelles et exporter la sélection. | Fondamentaux QGIS | 1 heure | 
| __[Exercice 3 : Sources de données](/content/fr/Module_2/fr_qgis_data_sources_ex3.md)__ | L’objectif de cet exercice est d’explorer différentes sources de données, de comprendre où et comment accéder à des données pertinentes, et d’identifier les problèmes potentiels. | Fondamentaux QGIS | 1 à 2 heures |
| __[Exercice 4 : Exporter des données OSM](/content/fr/Module_2/fr_qgis_data_sources_ex4.md)__ | L’objectif de cet exercice est d’apprendre plusieurs méthodes pour exporter des données OpenStreetMap (OSM) sous forme de fichiers vectoriels et les charger dans QGIS. | Acquisition et gestion des données | 2 heures |
| __[Exercice 5 : Le monde](/content/fr/Module_2/fr_qgis_module_2_ex_alt.md)__ | L’objectif de cet exercice est de créer un projet QGIS, d’importer des données vectorielles, de déplacer des couches dans la hiérarchie des couches et d’enregistrer les modifications apportées au projet QGIS. Cet exercice peut être réalisé seul à la maison. | Fondamentaux QGIS | 1 heure |
