# Ejercicios para el módulo 4

Estos ejercicios aluden brevemente a todo el contenido del módulo 4.

## Visualización

| Ejercicio | Descripción | Grupo focal | Duración estimada |
| :-------------------- | :----------------- |:----------------- |:----------------- |
| __[Ejercicio 1: Creación de un mapa de Ghana](/content/es/Module_4/es_qgis_map_design_I_ex1.md)__ | El objetivo de este ejercicio es crear un mapa general de Ghana con sus sub-distritos, sus carreteras principales, sus asentamientos y sus hospitales. Esta información resulta muy útil para el trabajo humanitario. La primera parte del ejercicio abarcará la Simbolización de las capas de datos. La segunda parte se centrará en crear el diseño de impresión. | Nivel intermedio de QGIS | 5 horas |
| __[Ejercicio 2: Creación de un mapa de situación de inundaciones en Larkana (Pakistán)](/content/es/Module_4/es_qgis_map_design_I_ex2.md)__ | El objetivo de este ejercicio es crear un mapa de la situación de las inundaciones en Larkana (Pakistán). Esta información resulta muy útil para el trabajo humanitario y especialmente, para la respuesta ante inundaciones. La primera parte del ejercicio abarcará la Simbolización de las capas de datos. La segunda parte se centrará en crear el diseño de impresión. | Nivel intermedio de QGIS | 3 horas |
| __[Ejercicio: Visualización de los resultados del impacto del ciclón: Aina aplica estilos a sus capas](/content/es/Module_4/es_module_4_mdg_aa_ex_4.md)__ | Este ejercicio es la tercer tarea del programa de ejercicios “Análisis de acciones anticipatorias para ciclones en Madagascar”. En este ejercicio, aplicaremos archivos de estilo a las salidas del modelo. | Pestaña de simbología y archivos de estilos | 2 horas |
| __[Ejercicio: Creación rápida de mapas: Aina utiliza plantillas de mapas](/content/es/Module_4/es_module_4_mdg_aa_ex_5.md)__ | Este ejercicio es la tercer tarea del programa de ejercicios “Análisis de acciones anticipatorias para ciclones en Madagascar”. En este paso, utilizaremos plantillas de mapas para crear mapas listos para imprimir. | Compositor de diseño de impresión y archivos de plantillas del mapa | 1 hora |
